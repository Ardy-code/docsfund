title: System Requirement

## Client Side Requirement
kebutuhan sistem yang harus dipenuhi oleh perangkat dari si pengguna:

**perangkat**

- Komputer, laptop atau perangkat seluler dengan spesifikasi minimum:
  - **Ram:** 2 GB atau lebih
  - **Sistem operasi:** Windows 10, macOS Mojave, Android 8.0, atau iOS 13 (Atau versi lebih baru).

**Browser**

Browser Modern Seperti: 

- **Google Chrome** (versi 90 ke atas).
- **Mozilla Firefox** (versi 88 ke atas).
- **Microsoft Edge** (versi chronium).


## Server Side Requirement
Kebutuhan sistem yang harus dipenuhi oleh server untuk menjalankan aplikasi

**Perangkat**

- **Prosesor:** Dual-Core 2.4 GHz atau lebih tinggi
- **Ram:** 4 GB atau lebih
- **Penyimpanan:** 20 GB ruang kosong (disarankan Menggunakan SSD untuk performa lebih baik).
- **Koneksi Internet:** Kecepatan upload dan download minimum 10 Mbps untuk mendukung akses pengguna simultan.

**Perangkat Lunak**

- **Sistem Operasi Server:** Ubuntu 20.04 atau CentOS 8.
- **Server Aplikasi:** Apache 2.4 atau Nginx.
- **Bahasa Pemrogaman Backend:** PHP (versi 7.4 ke atas)
- **Basis Data:** MYSQL (versi 8.0 ke atas)

## Development Environment Requirements

Persyaratan perangkat dan perangkat lunak yang diperlukan oleh pengembang untuk membangun aplikasi:

**IDE/Editor Kode**
- Visual Studio Code

**Perangkat Lunak Pendukung**
- Node.js untuk pengelolaan dependensi frontend.
- XAMPP untuk menjalankan server lokal

## Development Environment Requirements

Persyaratan perangkat dan perangkat lunak yang diperlukan oleh pengembang untuk membangun aplikasi:

**Platform Hosting**
- Vercel untuk deployment frontend.
- Hosting berbayar (opsional) untuk backend dengan dukungan PHP dan MySQL.
