Berisikan tentang deskripsi singkat dari **"Kosfund"**

Aplikasi **"Kosfund"** ialah aplikasi berbasis web untuk pencatatan keuangan. aplikasi ini di ciptakan untuk membantu para pengguna untuk mencatat dan mengatur keuangan mereka, meliputi pemasukan, dan pengeluaran. salah satu fitur rencana yang ingin di kembangkan dalam aplikasi kosfund ini ialah sistem kecerdasan buatan (AI). 
dimana dalam sistem AI akan sangat membantu pengguna untuk membuat Transaksi otomatis yang dimana jika tidak sesuai dapat di edit kembali.

Namun, pada fase pengembangan ini, fokus utamanya adalah menyediakan fitur inti, yaitu:

- membuat catatan keuangan
- menghapus catatan keuangan
- mengedit catatan keuangan

Fitur ini diharapkan menjadi fondasi utama sebelum mengintegrasikan fungsi yang lebih kompleks di masa depan.
