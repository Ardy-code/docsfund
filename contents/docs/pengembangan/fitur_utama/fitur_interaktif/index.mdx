## 1. Fitur Untuk Memasuki Mode Gelap
 Fitur Mode Gelap akan membantu pengguna untuk melihat tampilan web dengan keadaan yang tidak silau.

## 2. Fitur Pencarian Transaksi
Fitur ini dapat membantu catatan transaksi mereka tanpa perlu scroll. fitur ini dibuat agar ketika pengguna melakukan banyak transaksi dan merasa malas transaksi, mereka dapat mencari nya melalui fitur tersebut
